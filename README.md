# gheistand.dev

Personal and professional website for Glenn Heistand, hosted on Cloudflare Pages.

## About

This site serves as a central hub for my personal projects in engineering consulting, water resources engineering, flood risk management, and faith-based content. It includes:

- Personal portfolio and professional information
- BaseFlood Engineering, PLLC business presence
- Links to related projects and resources

## What's Included

- **Main Site**: Personal and professional landing page
- **BaseFlood Section**: Draft Information about BaseFlood Engineering, PLLC - FEMA flood risk consulting services
- **ElevenLabs AI Agent**: Interactive assistant for site visitors
- **Analytics**: Google Analytics integration for site metrics

## Technology Stack

- Static HTML/CSS/JavaScript
- Hosted on Cloudflare Pages
- Cloudflare Zero Trust for authentication
- ElevenLabs AI integration
- Google Analytics

## Deployment

This site is automatically deployed via Cloudflare Pages. Any push to the `main` branch triggers an automatic build and deployment.

### Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. Make changes and test locally
4. Commit and push to trigger deployment
